#  Neovim Master Cheatsheet (Split Edition)

**Основная философия:** `Пробел` — твой лучший друг (Leader). Кисти остаются на домашнем ряду (Home Row). Тянуться к `Esc` и стрелкам — незаконно.

## 1. Ядро (Core & Insert Mode)

Всё начинается здесь. Базовые рефлексы для сохранения фокуса и минимизации движений.

|**Клавиша**|**Режим**|**Действие**|
|---|---|---|
|`Space`|Normal|**Leader Key** (Префикс для всех кастомных команд)|
|`jj`|Insert|Моментальный выход из режима вставки (вместо `Esc`)|
|`<leader> h`|Normal|Сбросить подсветку после поиска (`:nohlsearch`)|

## 2. Окна и Сплиты (Window Management)

Идеально для координации левой и правой руки. Используй `Ctrl` на противоположной стороне от нажимаемой буквы `hjkl`.

|**Клавиша**|**Действие**|**Примечание для Split**|
|---|---|---|
|`:vsp`|Вертикальный сплит|Разделить экран пополам|
|`:sp`|Горизонтальный сплит|Разделить экран сверху/снизу|
|`Ctrl` + `h`|Фокус влево|Плавный переход в левое окно|
|`Ctrl` + `l`|Фокус вправо|Плавный переход в правое окно|
|`Ctrl` + `j`|Фокус вниз||
|`Ctrl` + `k`|Фокус вверх||

## 3. Буферы (BufferLine)

Навигация по открытым файлам без использования громоздких вкладок.

|**Клавиша**|**Действие**|
|---|---|
|`Shift` + `h`|Предыдущий буфер (шаг влево)|
|`Shift` + `l`|Следующий буфер (шаг вправо)|
|`Ctrl` + `q`|Закрыть текущий буфер (`:bdelete`)|

## 4. Поиск и Задачи (Telescope & Todo-comments)

Навигация по кодовой базе и отслеживание технического долга.

|**Клавиша**|**Действие (Telescope)**|
|---|---|
|`<leader> ff`|Найти файл в проекте (Find Files)|
|`<leader> fg`|Поиск по тексту во всех файлах (Live Grep)|
|`<leader> fb`|Список открытых буферов|
|`]t`|Прыгнуть на следующий TODO в текущем файле|
|`[t`|Прыгнуть на предыдущий TODO|
|`<leader> ft`|Открыть все TODO в проекте (Telescope)|

## 5. Инструменты (Терминал и Форматирование)

Интеграция твоего DevOps/QA воркфлоу прямо в редактор.

|**Плагин**|**Клавиша**|**Действие**|
|---|---|---|
|**Toggleterm**|`Ctrl` + `\` _(или твой бинд)_|Открыть/Скрыть плавающий терминал (для `git`, `docker`, `kubectl`)|
|**Conform**|`<leader> fm`|Запустить форматирование текущего файла (Format)|
|**Neoscroll**|`Ctrl` + `d` / `u`|Плавная прокрутка вниз/вверх (бережет глаза при чтении логов)|

## 6. Базовые Vim Motions (Для тренировки мышечной памяти)

На сплит-клавиатуре эти комбинации нужно довести до автоматизма.

|**Комбо**|**Описание**|**Зачем тренировать на сплите**|
|---|---|---|
|`ciw`|Change Inner Word (Удалить слово и войти в Insert)|Синхронизация разных пальцев `c` (лев) -> `i` (прав) -> `w` (лев)|
|`di(`|Удалить всё внутри скобок `()`|Тренировка слоя спецсимволов|
|`f` + `символ`|Найти символ в строке и прыгнуть к нему|Отучает спамить `l` или `w`|
|`V` + `jj` + `=`|Выделить строки и выровнять отступы|Базовое форматирование без плагинов|
